Live Demo

This demo runs the current portfolio version of Imago English – a modular Flask application that combines spaced-repetition flashcards, an audiobook reader with inline translation, and teacher/admin tooling.

How to explore quickly

  • On the landing page, use the demo login option to log in as test users (student, teacher, and admin) – no Google account required.
  • As a student, add a few flashcards and start a study session to see the custom spaced-repetition logic.
  • Switch to a teacher user to view the dashboard with assigned students and manage their flashcards.

What this demo shows

  • Role-based access: students, teachers, admins with different dashboards.
  • Flashcard engine using a customized SM-2 spaced-repetition algorithm with streaks and points.
  • Audiobook reader with text selection → Google Translate → “Add as flashcard”.
  • Leaderboard based on total points × study streak.
  • Calendar view using Google Calendar free/busy + per-teacher settings.

Tech Highlights

The app is written as a clean, portfolio-ready Flask project using the app-factory pattern, blueprints, and a documented development process.

  • Flask blueprints for auth, flashcards, audiobook, dashboard, admin, progress, and calendar.
  • SQLAlchemy models for users, flashcards, and calendar settings.
  • Google OAuth via Flask-Dance + Flask-Login session management.
  • Separate docs for architecture, API endpoints, frontend, and spaced-repetition logic.
